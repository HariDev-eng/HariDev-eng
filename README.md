## 🚀 Projects I'm proud of

### 🏥 [Patient Management Microservice System](https://github.com/HariDev-eng/Patient-Management-System)
A 6-service microservices architecture built with Spring Boot — patient-service, auth-service, billing-service, analytics-service, api-gateway, and integration-test. Services communicate via **Kafka** (async events) and **gRPC** (inter-service calls). Everything runs through Docker Compose.

`Spring Boot` `Kafka` `gRPC` `Docker` `PostgreSQL`

---

### 🎬 [Concurrent Movie Seat Booking System](https://github.com/HariDev-eng/concurrent-movie-booking)
A Go backend built to solve the hard problem of concurrent seat booking — preventing double-booking under 100,000 simultaneous requests. Implemented three storage strategies: in-memory, mutex-protected, and Redis-backed with atomic SET NX.

`Go` `Redis` `Goroutines` `Clean Architecture` `REST APIs`

---

### 📝 [Blog Platform](https://github.com/HariDev-eng/Blog-App)
A Spring Boot blog backend with JWT authentication, role-based access control via Spring Security, and full CRUD for posts, categories, and tags on PostgreSQL.

`Spring Boot` `Spring Security` `JWT` `PostgreSQL` `REST APIs`
